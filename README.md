## 📖 Current Readings

### Data Structures and Algorithms with JavaScript (Michael McMillan)

This section contains my implementations, notes, and solutions to the exercises proposed in the book. The goal is to master core computer science concepts using JavaScript.

#### 📁 Structure
The exercises are organized chronologically by chapters to maintain a clear learning path:

*   `ch01-javascript-programming-environment/` - Review of JS concepts, arrays, and environment setup.
*   `ch02-arrays/` - Built-in array architecture and custom implementations.
*   `ch03-lists/` - Designing and building a custom List class.
*   `ch04-stacks/` - Stack data structure applications (e.g., palindrome checking, recursion).
*   `ch05-queues/` - Queues, Priority Queues, and radix sorting simulation.
*   `ch06-linked-lists/` - Singly, Doubly, and Circularly Linked Lists.
*   `ch07-dictionaries/` - Key-value pair implementations.
*   `ch08-hashes/` - Hash tables, collision resolution (separate chaining & linear probing).
*   `ch09-sets/` - Set operations (union, intersection, subset, difference).
*   `ch10-binary-trees/` - Binary Search Trees (BST) traversal and search algorithms.
*   `ch11-graphs/` - Graph representations, BFS, and DFS algorithms.
*   `ch12-sorting-algorithms/` - Bubble, Selection, Insertion, Shellsort, Mergesort, and Quicksort.
*   `ch13-searching-algorithms/` - Sequential and Binary Search.
*   `ch14-advanced-algorithms/` - Dynamic Programming and Greedy Algorithms.
